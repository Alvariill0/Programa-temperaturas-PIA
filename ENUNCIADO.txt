
Partiendo de nuestro ejemplo anterior vamos a obtener temperaturas de la capital de todos los países de Europa y trabajar con ellos.

Para ello:

1.	Almacenar en una base de datos MySQL los siguientes datos de todos los países del fichero obtenido en la práctica anterior (Europa):
a.	Nombre
b.	Capital
c.	Región
d.	Subregión
e.	Código cca2
f.	Código cca3
g.	Si pertenece a la Naciones Unidas (campo unMember)
h.	Fronteras

2.	Obtener temperaturas para todas las capitales de los países europeos. 
Para ello vamos a usar el API indicado más abajo.

Usaremos el formato del API tanto en formato XML como en formato JSON. Para ello dividid los resultados de las capitales y solicitad la mitad en un formato y la mitad en otro de forma que podáis hacer el tratamiento de las dos formas.

3.	Extraer los siguientes datos de las temperaturas obtenidas y almacenarlos en la BD asociados al timestamp del momento obtenido de forma que cuando hagáis varias llamadas tengáis todas ellas.


a.	temperatura, 
b.	sensación térmica 
c.	mínima 
d.	máxima
e.	humedad
f.	hora de amanecer 
g.	hora de atardecer

La obtención de las temperaturas se debe poder realizar más de una vez. Para ello nuestro programa debe tener una opción que nos permita ejecutar este proceso para todas las capitales.

4.	Realizar una aplicación que pida el nombre de un país y muestre los datos de temperatura más actuales que haya en la base de datos de su capital y de las capitales de sus países fronterizos. 

5.	El menú de la aplicación debe ser:
-1.	Insertar datos de países desde JSON (este punto sólo lo ejecutaré una vez)
-2.	Obtener temperaturas de la API para todos los países y guardarlas en base de datos (se podrá ejecutar más de una vez)
-3.	Obtener de la base de datos la temperatura de un país y sus fronterizos a partir de país dado por usuario.
